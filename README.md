# ConsoleCalc
Basic calculator for console

# Build
Use cmake for build

# Using calc
Just enter your expression. Calc supports floating point numbers(c-type double).
You can use following examples to enter numbers:
`42`	`3.27`

Calculator supports following operators:
`+ - / * % ^ ! ( )`
